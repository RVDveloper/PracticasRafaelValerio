PROYECTOS

AnafaelBeats MVC:

Es una aplicación web desarrollada en C# utilizando el patrón de diseño Modelo-Vista-Controlador (MVC) e utilizando la comunicacion asincronica AJAX. Inspirada en servicios de streaming de música como Spotify, Anafael Beats MVC busca proporcionar una experiencia completa para la exploración, reproducción y búsqueda de música y letras de canciones Mediante el consumo de APIs

AnafaelBeats:

Este proyecto es una aplicación web de música similar a Spotify, construida utilizando Flask, MySQL, JavaScript y APIs. La aplicación permite a los usuarios explorar y reproducir música, crear listas de reproducción y obtener recomendaciones basadas en las Tendencias.

Flask ApiAnimeWeb:

He creado una API sencilla con Flask que permite a los usuarios ver capítulos de anime. Este proyecto representa mis primeros pasos en el desarrollo de APIs utilizando Flask, y me ha servido como experiencia de aprendizaje para comprender los fundamentos.
